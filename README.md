---

## 使用方法

在你的 Unity 项目的 `Packages/manifest.json` 中添加依赖：

```json
{
  "dependencies": {
    "com.snikit.csg": "https://github.com/CS-LX/CSG---3d-boolean-operations-Unity-.git"
  }
}
```

> 你也可以用 tag 来锁定版本，比如 `#1.0.0`。
